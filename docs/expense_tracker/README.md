Описание
GUI-приложение для учёта личных расходов. Позволяет добавлять транзакции, фильтровать их по категории и периоду, автоматически подсчитывать сумму расходов за выбранный промежуток времени. Данные сохраняются и загружаются в формате JSON.
Как использовать
1. Убедитесь, что установлен Python 3.7+
2. Запустите приложение: `python main.py`
3. Заполните поля: сумма, категория, дата (формат ГГГГ-ММ-ДД)
4. Нажмите "➕ Добавить расход"
5. Для анализа используйте фильтры "Период с/по" и "Категория"
6. Сумма автоматически пересчитывается под текущий фильтр
7. Данные можно сохранить/загрузить через кнопки внизу окна
Технологии
- Python 3
- tkinter (GUI)
- json (хранение данных)
- Git (версионирование)
